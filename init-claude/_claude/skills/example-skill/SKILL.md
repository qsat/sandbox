# Example Skill

This is a template for creating custom skills.

## When to Use

Describe the trigger conditions. Example:
- When user asks to "do X"
- When user types `/skill-name`

## Steps

1. First, check the current state: `<command>`
2. Then perform the main action
3. Verify the result

## Notes

- Keep skills focused on a single, reusable workflow
- Skills are invoked via the `Skill` tool
- Document the expected inputs and outputs
